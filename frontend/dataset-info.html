Dansbart Open Dataset

Ett öppet dataset med audio-analys och klassificering av svensk folkmusik, inklusive mänsklig feedback och ground truth data.

Detta dataset kombinerar automatisk analys från neckenml-analyzer med validering och korrigeringar från Dansbart-användare. Perfekt för träning av machine learning-modeller för musikanalys.

Vad ingår i datasetet?

Audio-analysdata

  • swing_ratio - Swingkänsla (0.0-1.0)
  • articulation - Hur tydligt noterna är separerade
  • bounciness - Studskänsla och energi
  • loudness - EBU R128 Loudness (LUFS)
  • punchiness - Transient score för energiska anslag
  • voice_probability - Sannolikhet för sång
  • polska_score - Asymmetrisk rytm-score för polska
  • hambo_score - Asymmetrisk rytm-score för hambo
  • bpm_stability - 1.0 = metronomisk, 0.0 = rubato

Klassificeringar

  • Dansstil (t.ex. "Polska", "Vals", "Schottis") med konfidenspoäng
  • Understil (t.ex. "Slängpolska", "Bingsjöpolska")
  • Musikgenre (traditional_folk, modern_folk, folk_pop, contemporary)
  • Tempo-kategorier och effektiv BPM
  • Antal användarbekräftelser per klassificering

Mänsklig feedback (Ground Truth)

  • Stilkorrigeringar från användare
  • Rörelsetaggar (t.ex. "Sviktande", "Tungt")
  • Konsensusdata om dansstilar
  • Användarvaliderade strukturannoteringar

Låtstruktur

  • Taktpositioner (bars)
  • Sektioner (sections)
  • Sektionsetiketter (A, B, C, etc.)

Fokus på analysdata

Detta dataset innehåller Dansbarts egna analyser och metadata:

  • Audio-analys från neckenml-analyzer
  • Dansstilklassificeringar med konfidenspoäng
  • Mänsklig feedback och ground truth-data
  • Låtstruktur-annoteringar
  • Musikgenre-klassificeringar
  • Allmänt tillgänglig metadata (ISRC, artistnamn, låttitlar)

Vi inkluderar endast data som vi själva har genererat eller som är allmänt tillgänglig. Plattformsspecifika identifierare och länkar ingår inte för att respektera tredjepartsleverantörers användarvillkor.

Användningsområden

ML-modellträning

Träna och validera modeller för musikklassificering, särskilt för folkmusik och svenska danstyper.

Forskning

Analysera egenskaper hos svensk folkmusik, rytmiska mönster, eller muzikinstrumentanalys.

Verktygs­utveckling

Bygg kompletterande verktyg, appar eller tjänster för folkmusik- communityn baserat på ground truth-data.

Benchmark

Använd som benchmark-dataset för att utvärdera och jämföra olika musikanalys-algoritmer.
