Integritetspolicy

Senast uppdaterad: 10 december 2025

Introduktion

Välkommen till Dansbart.se! Vi hjälper dig hitta dansmusik baserat på dansstil och tempo. Denna integritetspolicy förklarar hur vi samlar in, använder och skyddar din information i enlighet med Dataskyddsförordningen (GDPR) och svensk lag.

Vi värdesätter din integritet och arbetar för att samla in så lite data som möjligt. Vi säljer aldrig dina uppgifter och delar dem inte med tredje part utöver de tjänster som krävs för att tillhandahålla musikuppspelning.

Registeransvarig

Dansbart.se är ansvarig för behandlingen av dina personuppgifter.

Kontakt för dataskyddsärenden:
E-post: privacy@dansbart.se
Vi svarar på förfrågningar inom 30 dagar enligt GDPR.

Vilken data vi samlar in

1. Lokal lagring i din webbläsare (localStorage)

Följande data lagras lokalt på din enhet och skickas aldrig till vår server:

  • cookie_consent_v1 - Ditt samtycke för cookies (status, tidsstämpel, version)
  • structure_vote_v1_* - Flaggor för att komma ihåg om du röstat på låtstrukturer
  • fb_* - Flaggor för att undvika upprepad feedback på samma låt

Observera: Denna data stannar på din enhet. Du kan när som helst radera den genom att rensa webbläsarens localStorage eller använda privat surfläge.

2. Anonym feedback till vår server

När du skickar in korrigeringar eller förslag sparar vi följande anonyma data:

  • Dansstilskorrigeringar (t.ex. "Detta är en vals, inte en polska")
  • Tempojusteringar och tempokategori
  • Strukturredigeringar (taktslag, sektioner)
  • Känslomässiga taggar och rörelsebeskrivningar
  • Länkar till Spotify eller YouTube för en låt
  • Rapporter om brutna länkar

Ingen identifierande information sparas. Vi samlar inte in IP-adresser, användar-ID:n, eller annan data som kan kopplas till dig personligen. Feedback är helt anonym och används enbart för att förbättra våra algoritmer och musikkatalogen.

3. Tredjepartscookies

När du ger samtycke och spelar musik använder vi följande tredjepartstjänster:

Spotify

När du spelar musik via Spotify laddar vi Spotifys inbäddade spelare, som kan sätta cookies och samla in användningsdata enligt Spotifys egen integritetspolicy.

Läs Spotifys integritetspolicy →

YouTube (Google)

När du spelar musik via YouTube laddar vi YouTubes inbäddade spelare, som kan sätta cookies och samla in användningsdata enligt Googles integritetspolicy.

Läs Googles integritetspolicy →

Viktigt: Vi laddar aldrig dessa tredjepartstjänster förrän du har gett ditt samtycke via vår cookie-banner. Du kan när som helst ändra ditt samtycke via "Cookie-inställningar" i sidfoten.

Hur vi använder din data

Vi använder insamlad data för att:

  • Förbättra klassificeringsalgoritmer för dansstilar och tempo
  • Lära av gemenskapsbaserade korrigeringar och förslag
  • Tillhandahålla musikuppspelning via Spotify och YouTube
  • Undvika att visa samma feedback-förfrågningar upprepade gånger
  • Komma ihåg dina cookie-preferenser

All feedback används för att förbättra tjänsten för alla användare. Vi bygger ingen personlig profil av dig och spårar inte ditt beteende mellan olika sessioner.

Rättslig grund för behandling

Vi behandlar data baserat på följande rättsliga grunder:

  • Samtycke (GDPR Artikel 6.1a): För tredjepartscookies från Spotify och YouTube
  • Berättigat intresse (GDPR Artikel 6.1f): För anonym feedback som förbättrar tjänsten

Eftersom all feedback är anonym och inte kan kopplas till individer, bedömer vi att vårt berättigade intresse att förbättra tjänsten väger tyngre än eventuella integritetsrisker.

Dina rättigheter enligt GDPR

Du har följande rättigheter:

  • Rätt till tillgång (Artikel 15): Du kan begära en kopia av den data vi har om dig
  • Rätt till radering (Artikel 17): Du kan begära att vi raderar dina uppgifter
  • Rätt till dataportabilitet (Artikel 20): Du kan begära att få dina uppgifter i maskinläsbart format
  • Rätt att återkalla samtycke (Artikel 7.3): Du kan när som helst återkalla ditt cookie-samtycke
  • Rätt att göra invändningar (Artikel 21): Du kan invända mot behandling baserad på berättigat intresse

Observera: Eftersom vi inte samlar in personuppgifter eller IP-adresser för feedback, kan vi inte identifiera eller radera specifika feedback-bidrag. All feedback är anonym från början.

För att utöva dina rättigheter, kontakta oss på: privacy@dansbart.se

Datadelning och tredje parter

Vi säljer aldrig dina uppgifter. Vi delar inte din data med någon tredje part, förutom:

  • Spotify och YouTube: När du ger samtycke och spelar musik
  • Webbhotell: Vår server är värd hos en leverantör som följer GDPR

Vi använder för närvarande ingen analystjänst (t.ex. Google Analytics). Om vi i framtiden väljer att lägga till analytics kommer vi att uppdatera denna policy och be om ditt samtycke.

Datalagring och radering

  • Cookie-samtycke: Sparas i 13 månader, därefter upphör det att gälla och måste förnyas
  • Lokal lagring (localStorage): Sparas tills du rensar webbläsarens cache eller använder privat surfläge
  • Anonym feedback: Sparas på obestämd tid i vår databas för att förbättra algoritmer

Eftersom feedback är anonym behöver den inte raderas enligt GDPR:s principer om lagringsbegränsning (Artikel 5.1e), då den inte kan kopplas till en identifierad person.

Säkerhet

Vi vidtar rimliga tekniska och organisatoriska åtgärder för att skydda dina uppgifter:

  • All kommunikation sker över krypterad HTTPS-anslutning
  • Vi samlar inte in känsliga personuppgifter
  • Feedback är designad att vara anonym från början
  • Vi har ingen inloggning eller användarhantering, vilket minskar risker

Ändringar av denna policy

Vi kan komma att uppdatera denna integritetspolicy från tid till annan. Vid väsentliga ändringar kommer vi att:

  • Uppdatera datumet "Senast uppdaterad" högst upp på denna sida
  • Visa en notis på webbplatsen om ändringarna
  • Be om förnyat samtycke om det krävs enligt lag

Vi rekommenderar att du regelbundet granskar denna policy för att hålla dig informerad.
