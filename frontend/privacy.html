Integritetspolicy

Senast uppdaterad: 12 december 2025

Introduktion

Välkommen till Dansbart.se! Vi hjälper dig hitta dansmusik baserat på dansstil och tempo. Denna integritetspolicy förklarar hur vi samlar in, använder och skyddar din information i enlighet med Dataskyddsförordningen (GDPR) och svensk lag.

Vi värdesätter din integritet och arbetar för att samla in så lite data som möjligt. Vi säljer aldrig dina uppgifter och delar dem inte med tredje part utöver de tjänster som krävs för att tillhandahålla musikuppspelning.

Registeransvarig

Dansbart.se är ansvarig för behandlingen av dina personuppgifter.

Kontakt för dataskyddsärenden:
E-post: privacy@dansbart.se
Vi svarar på förfrågningar inom 30 dagar enligt GDPR.

Vilken data vi samlar in

1. Lokal lagring i din webbläsare (localStorage)

Följande data lagras lokalt på din enhet:

  • cookie_consent_v1 - Ditt samtycke för cookies (status, tidsstämpel, version) - skickas aldrig till servern
  • structure_vote_v1_* - Flaggor för att komma ihåg om du röstat på låtstrukturer - skickas aldrig till servern
  • fb_* - Flaggor för att undvika upprepad feedback på samma låt - skickas aldrig till servern
  • analytics_session_id - Slumpmässigt genererat sessions-ID - skickas till servern för att identifiera återkommande besökare i statistiksyfte
  • analytics_returning_visitor - Flagga som indikerar om du är en återkommande besökare - används endast lokalt

Observera: Du kan när som helst radera all denna data genom att rensa webbläsarens localStorage eller använda privat surfläge. Sessions-ID:t är helt slumpmässigt och kan inte användas för att identifiera dig personligen.

2. Anonym feedback till vår server

När du skickar in korrigeringar eller förslag sparar vi följande anonyma data:

  • Dansstilskorrigeringar (t.ex. "Detta är en vals, inte en polska")
  • Tempojusteringar och tempokategori
  • Strukturredigeringar (taktslag, sektioner)
  • Känslomässiga taggar och rörelsebeskrivningar
  • Länkar till Spotify eller YouTube för en låt
  • Rapporter om brutna länkar

Ingen identifierande information sparas. Vi samlar inte in IP-adresser, användar-ID:n, eller annan data som kan kopplas till dig personligen. Feedback är helt anonym och används enbart för att förbättra våra algoritmer och musikkatalogen.

3. Anonym användningsanalys (Analytics)

Vi använder ett egenbyggt analytics-system för att förstå hur Tjänsten används och för att förbättra användarupplevelsen. Följande anonyma data samlas in:

  • Sessionsdata: Slumpmässigt genererat sessions-ID (lagrat lokalt i din webbläsare), tidsstämplar för första och senaste besök, antal sidvisningar, användar-agent (webbläsare och enhet)
  • Uppspelningsstatistik: Vilka låtar som spelades, vilken plattform (YouTube/Spotify), hur länge du lyssnade, om du lyssnade färdigt (>30 sekunder)
  • Interaktionsdata: När du ser eller interagerar med feedback-förfrågningar, när du öppnar eller stänger modaler, vilken typ av rapporter du skickar in
  • Besöksmönster: Tidpunkt för besök (timme på dygnet, datum) för att förstå trafikflöden

Viktig information om analytics:

  • Inga IP-adresser: Vi loggar eller sparar aldrig din IP-adress
  • Inget spårning mellan enheter: Sessions-ID genereras slumpmässigt och är unikt per webbläsare
  • Inget säljande av data: Analytics-data används endast internt för att förbättra Tjänsten
  • Ingen tredjepartsanalys: Vi använder INTE Google Analytics, Facebook Pixel eller andra externa analytics-tjänster
  • Anonymiserat: All data är anonym och kan inte kopplas till en specifik person

Sessions-ID:t används för att identifiera om du är en återkommande besökare (för statistik), men det kan inte användas för att identifiera dig personligen. Om du rensar webbläsarens localStorage eller använder privat surfläge kommer ett nytt sessions-ID att genereras.

4. Tredjepartscookies

När du ger samtycke och spelar musik använder vi följande tredjepartstjänster:

Spotify

När du spelar musik via Spotify laddar vi Spotifys inbäddade spelare, som kan sätta cookies och samla in användningsdata enligt Spotifys egen integritetspolicy.

Läs Spotifys integritetspolicy →

YouTube (Google)

När du spelar musik via YouTube laddar vi YouTubes inbäddade spelare, som kan sätta cookies och samla in användningsdata enligt Googles integritetspolicy.

Läs Googles integritetspolicy →

Viktigt: Vi laddar aldrig dessa tredjepartstjänster förrän du har gett ditt samtycke via vår cookie-banner. Du kan när som helst ändra ditt samtycke via "Cookie-inställningar" i sidfoten.

Hur vi använder din data

Vi använder insamlad data för att:

  • Förbättra klassificeringsalgoritmer för dansstilar och tempo
  • Lära av gemenskapsbaserade korrigeringar och förslag
  • Tillhandahålla musikuppspelning via Spotify och YouTube
  • Undvika att visa samma feedback-förfrågningar upprepade gånger
  • Komma ihåg dina cookie-preferenser
  • Förstå användningsmönster: Analysera vilka låtar som är populära, vilka plattformar som föredras, och när besökare använder tjänsten
  • Förbättra användarupplevelsen: Identifiera vilka funktioner som används och vilka som kan förbättras
  • Mäta engagemang: Förstå hur användare interagerar med tjänsten (t.ex. hur många som lyssnar färdigt på låtar, hur många som bidrar med feedback)

All feedback och analytics-data används för att förbättra tjänsten för alla användare. Vi bygger ingen personlig profil av dig och all data är anonym och aggregerad.

Rättslig grund för behandling

Vi behandlar data baserat på följande rättsliga grunder:

  • Samtycke (GDPR Artikel 6.1a): För tredjepartscookies från Spotify och YouTube
  • Berättigat intresse (GDPR Artikel 6.1f): För anonym feedback och analytics som förbättrar tjänsten

Eftersom all feedback och analytics-data är anonym och inte kan kopplas till individer, bedömer vi att vårt berättigade intresse att förbättra tjänsten väger tyngre än eventuella integritetsrisker. Vi har genomfört en intresseavvägning enligt GDPR och bedömer att den anonyma karaktären av databehandlingen innebär minimal risk för den registrerades rättigheter och friheter.

Dina rättigheter enligt GDPR

Du har följande rättigheter:

  • Rätt till tillgång (Artikel 15): Du kan begära en kopia av den data vi har om dig
  • Rätt till radering (Artikel 17): Du kan begära att vi raderar dina uppgifter
  • Rätt till dataportabilitet (Artikel 20): Du kan begära att få dina uppgifter i maskinläsbart format
  • Rätt att återkalla samtycke (Artikel 7.3): Du kan när som helst återkalla ditt cookie-samtycke
  • Rätt att göra invändningar (Artikel 21): Du kan invända mot behandling baserad på berättigat intresse

Observera: Eftersom vi inte samlar in personuppgifter eller IP-adresser för feedback eller analytics, kan vi inte identifiera eller radera specifika bidrag eller sessiondata kopplat till en viss person. All data är anonym från början. Du kan dock när som helst radera sessions-ID:t från din webbläsares localStorage för att starta en ny session.

För att utöva dina rättigheter, kontakta oss på: privacy@dansbart.se

Datadelning och tredje parter

Vi säljer aldrig dina uppgifter. Vi delar inte din data med någon tredje part, förutom:

  • Spotify och YouTube: När du ger samtycke och spelar musik
  • Webbhotell: Vår server är värd hos en leverantör som följer GDPR

Vi använder ett egenbyggt analytics-system som lagrar all data på vår egen server. Vi använder INTE externa analystjänster som Google Analytics, Facebook Pixel, eller liknande tredjepartsverktyg. All analytics-data förblir hos oss och delas aldrig med externa parter.

Datalagring och radering

  • Cookie-samtycke: Sparas i 13 månader, därefter upphör det att gälla och måste förnyas
  • Lokal lagring (localStorage): Sparas tills du rensar webbläsarens cache eller använder privat surfläge
  • Anonym feedback: Sparas på obestämd tid i vår databas för att förbättra algoritmer
  • Analytics-data: Sparas på obestämd tid i vår databas för statistik och förbättringsändamål

Eftersom både feedback och analytics-data är anonym behöver den inte raderas enligt GDPR:s principer om lagringsbegränsning (Artikel 5.1e), då den inte kan kopplas till en identifierad person. Data som inte är personuppgifter omfattas inte av GDPR:s lagringstidsbegränsningar.

Säkerhet

Vi vidtar rimliga tekniska och organisatoriska åtgärder för att skydda dina uppgifter:

  • All kommunikation sker över krypterad HTTPS-anslutning
  • Vi samlar inte in känsliga personuppgifter
  • Feedback är designad att vara anonym från början
  • Vi har ingen inloggning eller användarhantering, vilket minskar risker

Ändringar av denna policy

Vi kan komma att uppdatera denna integritetspolicy från tid till annan. Vid väsentliga ändringar kommer vi att:

  • Uppdatera datumet "Senast uppdaterad" högst upp på denna sida
  • Visa en notis på webbplatsen om ändringarna
  • Be om förnyat samtycke om det krävs enligt lag

Vi rekommenderar att du regelbundet granskar denna policy för att hålla dig informerad.
